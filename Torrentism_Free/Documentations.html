How to Display Recent Posts (Torrents) via LABELs

To display recent posts via labels, make sure your have at least 3 posts published under a specific label with proper images uploaded by yourself from your hard drive. Note: Images used in the post from other sources might make this feature to not work the way you are wanting.

    1. Go to Blogger >> Layout >> Featured Post Area >> Select any Widget >> Edit
    2. In the text area enter your label name i.e. Games, PC Games, Arcade and etc.
    3. Save the Widget, Press "Save" Button

How To Add Seeds in Posts (Torrent):

Every Torrent has some seeds, so you can define different seeds for different torrents (posts):

    1. Go to Blogger >> Add new Post >> Labels
    2. In Labels enter seeds_124 (Where 124 is the no of seeds of the torrent).
    3. Once everything is done, press "Publish" and your torrent now has no of seeds to show to users

How To Add Download Button:

To Add Torrent Download Button Do as follows

    1. Go to Blogger >> Add new Post >> Revert to HTML TAB and paste the following Code:
    2. <div class="downloadb">
<a href="#link-here">Download Torrent</a>
</div>
    3. Once done, Publish post
